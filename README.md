# itDAGENE-webapp
> nextgen itDAGENE frontend written with cool stuff

Written using `next.js`, `react-relay` and `graphql`.

## Setup

```bash
$ yarn schema prod # or dev
$ yarn relay
```

## Run in dev

```bash
$ yarn dev
$ # open http://localhost:8000
```

## Build

```bash
$ yarn build
$ yarn start
```
