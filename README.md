Written using [next.js](https://github.com/zeit/next.js/), [react-relay](https://github.com/facebook/relay/) and [graphql](http://graphql.org/).

## Setup

```bash
$ yarn
$ yarn schema prod # or dev
$ yarn relay
```

## Development

```bash
$ yarn dev
$ # open http://localhost:8000
```

## Production

```bash
$ yarn build
$ yarn start
```
